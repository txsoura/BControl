BControl | Sales and stock maneger system / Sistema de gestão de vendas e estoque
===========================================================================================================================
How to proceed with the installation? / Como prosseguir com a instalação?
*********************************
To use the system, you must follow these steps: / Para usar o sistema, tem de seguir os seguintes passos:
•	Install JDK and Xampp which are in the “other” folder; / Instalar o JDK e Xampp que se encontram na pasta “other”;
•	Restart the computer; / Reiniciar o computador;
•	After rebooting, just open Xampp, and boot Apache and MySQL. / Após a reinicialização do computador, é só abrir o Xampp, e inicializar o Apache e MySQL.
======================================================================================================================================================================
How to configure MySQL? / Como configurar o MySQL?
************************
To configure MySQL, simply open the browser after launching Xampp (Apache and MySQL), and type "localhost / phpmyadmin" and follow these instructions: / Para configurar o MySQL, basta abrir o navegador apor iniciar o Xampp (Apache e MySQL), e digitar “localhost/phpmyadmin” e seguir as seguintes instruções:
•	Press "New" in the left side menu; / Pressionar “New” no menu lateral esquerdo;
•	Type “bcontrol” in the field Database name; / Escrever “bcontrol” no campo Database name;
•	Select “utf8_general_ci” in the collection field and press “create”; / Selecionar “utf8_general_ci” no campo collection e pressionar “create”;
•	Select the “import” option from the top menu; / Selecionar a opção “import” no menu superior;
•	In the “browse” option select the “bcontrol.sql” file in the “other” folder of the system and press “go”; / Na opção “browse” selecionar o ficheiro “bcontrol.sql” que se encontra na pasta “other” do sistema e pressionar “go”;


*After all the previous steps, just open the BControl application in order to use the system. / Após todos os passos anteriores, é só abrir o aplicativo BControl, para poder usar o sistema.
